Group Selector

-> Group Selector select using ','.

-> Group selector is use to select the multiple element of that webpage.

Here is an example,
<p></p>
<div></div>

apply style using internal css.

        <style>
        p, div {
            color: red;
            background: rgb(250, 183, 40);
            padding: 20px;
        }
        </style>
