Id Selector

-> Id Selector select using '#' and then id name.

-> Id is must be unique. We can't use same name of id in one webpage.

-> Can't apply multiple id in single element

Here is an example,
<div id="block"></div>

apply style using internal css.

Here is an id name is block.

        <style>
        #block {
            height: auto;
            background: lightskyblue;
        }
        </style>
