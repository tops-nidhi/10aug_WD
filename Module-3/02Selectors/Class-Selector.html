Class Selector

-> Class Selector select using '.' and then class name.

-> Don't need to be create an unique class name. We can use same name of class multiple time in one webpage.

-> Can apply multiple class in single element.

Here is an example,
<div class="contain"></div>

apply style using internal css.

Here is an class name is contain.

        <style>
        .contain {
            height: auto;
            background: lightgray;
            padding: 10px;
        }
        </style>
